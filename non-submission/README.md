# non-submission

Thư mục này chứa tài liệu và cấu hình phục vụ phát triển với AI (Cursor Agent). **Không nộp** cùng bài đồ án.

## Nội dung

| Mục | Mô tả |
| --- | --- |
| `docs/` | Đặc tả, kiến trúc, checklist, workflow Cursor |
| `AGENTS.md` | Quy tắc cho Cursor Agent |
| `README-AGENT.md` | Hướng dẫn bộ starter kit cho Agent |
| `manifest.json` | Manifest bộ tài liệu Cursor |
| `.cursor/` | Project rules cho Cursor |
| `starter-assets/` | Tài nguyên khởi tạo (placeholder SVG gốc) |

Khi nộp bài, loại trừ toàn bộ thư mục `non-submission/` khỏi file zip hoặc repository nộp thầy.
